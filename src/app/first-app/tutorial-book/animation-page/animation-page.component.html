Animation List and steps

Step 1: First import Browser Animation module in Appmodule.ts 
                import { BrowserAnimationsModule } from '@angular/platform-browser/animations';

Step 2: Add Animations and changes in Component.ts file 
                @Component({
                    selector: 'app-first-app-page',
                    templateUrl: './pages.component.html',
                    styleUrls: ['./pages.component.scss'],
                    animations: [ trigger('moveIn',[
                    state('void', style({ gridTemplateColumns:'0% 100%'})),
                    state('*', style({gridTemplateColumns: '15%, 85%'})),
                    transition('void <=> *',animate('70ms'))
                    ]) ]
                  })
            
                1. trigger is the function that takes name of trigger and an Array that contain metaData about Animation
                    i) State:  state is a function that contains state name and style function
                    ii) Animation:  Animation takes time of animation
